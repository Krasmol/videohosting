# Отладочная информация

## Проблема с "Мой канал"

### Что проверить:

1. **Откройте консоль браузера** (F12 → Console)
2. **Нажмите "Мой канал"** в меню
3. **Посмотрите вывод** в консоли:
   - `Current user:` - ваш user_id
   - `All channels:` - список всех каналов с author_id
   - `Looking for channel with author_id:` - какой ID ищем
   - `My channel:` - найденный канал или undefined

### Возможные причины:

1. **Типы данных не совпадают**:
   - user.id может быть числом (3)
   - channel.author_id может быть строкой ("3")
   - JavaScript `===` не найдет совпадение

2. **Канал создан под другим пользователем**:
   - Вы вошли как user_id=3
   - Но канал создан с author_id=2

### Временное решение:

Если в консоли видите что типы не совпадают, я исправлю сравнение на:
```javascript
const myChannel = channels.find(c => c.author_id == user.id); // == вместо ===
```

### Постоянное решение:

Нужно убедиться что API возвращает правильные типы данных (числа как числа, а не строки).
